## 基于Qt 6.8.2 开发的简易音乐播放器


## 功能特性

- 功能 1：实现基本 UI 然后完成布局和规范化命名。
- 功能 2：读取本地文件夹并双击播放音乐。
- 功能 3：实现切换逻辑，即播放完自动切下一首、上下曲按钮交互、正播放歌曲高亮跟踪。
- 功能 4：丰富控制逻辑，即控制音乐播放暂停、音量大小调节、以及拖动进度条和时长显示。
